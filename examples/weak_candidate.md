# Weak Candidate Example
Role: Data Analyst Intern
Focus: Technical

Interviewer: Tell me about a data analysis project you worked on.
Candidate: I used Python and made graphs for a project. It went well.
Interviewer: What data cleaning steps did you perform?
Candidate: Mostly removing null values and stuff like that.
Interviewer: How did you decide which analysis method to use?
Candidate: I just tried what seemed okay.
[continue 5-6 turns]

Summary:
- Vague answers
- Low specificity
- Weak analytical reasoning
- Final coaching: practice STAR structure, metrics, and method justification
